# sbt-class-diagram

### requirement

- <https://www.graphviz.org/>

### install

#### `build.sbt`

```scala
enablePlugins(ClassDiagramPlugin)
```

#### `project/plugins.sbt`

##### latest stable version

```scala
addSbtPlugin("com.github.xuwei-k" % "sbt-class-diagram" % "0.3.1")
```
